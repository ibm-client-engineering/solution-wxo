## Send files to ADP

Sends a file or files to an Autmation Document Processing (ADP) instance

WXO name "JC - Send a PDF to ADP"

:::warning
ADP currently only supports sending PDF files to ADP via the API. For out of the box Watsonx Orchestrate document processing (internal ADP), use this [documentation](https://www.ibm.com/docs/en/watson-orchestrate?topic=skills-building-document-processing)
:::
